Tool orchestration
T1
get_building_violations sequential · runs first
HPD Violations API (wvxf-dwi5). Returns open violations by class + constructs BBL for downstream tools.
→ Cache key: VIOLATIONS:{housenumber}:{streetname}:{borough}
→ BBL: boroid + block.zfill(5) + lot.zfill(4)
→ Also queries false certs: currentstatus='CERTIFICATION CLOSEOUT' AND currentstatusdate > 90_days_ago
T2
get_building_complaints parallel with T3
HPD Complaints API (ygpa-z7cr). Needs BBL from T1. Returns 12-month complaint trend and category breakdown.
→ Trend: if (last_3mo / 3) > (prior_9mo / 9) → trending up
→ Cache key: complaints:{bbl}
T3
get_building_owner parallel with T2
HPD Registration + Contacts. Cross-references Worst Landlord Watchlist using exact match + alias table.
→ Registration (tesw-yqqr) → get registrationid
→ Contacts (feu5-w2e2) → CorporateOwner + IndividualOwner + Agent
→ Alias table handles LLC variant names (e.g. "A&E" → "A & E REAL ESTATE HOLDINGS LLC")
T4
get_owner_portfolio instant · local JSON
Pre-computed portfolio stats from static JSON. Zero network latency. Runs after T3 returns owner name.
→ Portfolio JSON has top 100 watchlist landlords
→ Run scripts/build_cache.py before hackathon to populate with live data

3-minute demo script
Setup
0:00–0:30
Open the app on your phone. Set the context: "HPD inspectors walk into buildings with a clipboard and a single complaint. The landlord has a portfolio management system, lawyers, and shell companies. We gave the inspector an investigator."
Briefing
0:30–1:30
Tap mic. Speak:
"Brief me on 1520 Sedgwick Avenue, Bronx." Agent delivers the full briefing. Room hears: 47 violations, heat complaints, Koscal 59 LLC, number 3 on the Worst Landlord Watchlist, 29 buildings, 2,300 violations. Let the portfolio reveal land.
Contrast
1:30–2:00
Tap mic. Speak:
"Brief me on 515 Park Avenue, Manhattan." Agent: "Zero open violations. Zero complaints in the past year." Let the silence land. That's the inequality moment. Don't explain it.
Vision
2:00–2:45
Tap camera icon. Point phone at prepared photos on tablet. Agent classifies: "This looks like it could be a Class C active leak — does that match what you're seeing?" Inspector confirms. Agent drafts the violation through conversation.
Close
2:45–3:00
"That took two minutes. Without this, it takes hours — if it happens at all. The landlord has always had this information. Now the inspector does too."
